# Little Miss Violet Cafe Website

**Status**: Active client project  
**Client**: Little Miss Violet Cafe  
**Location**: Point Cook, Melbourne  
**Purpose**: Website development for local cafe business  

## Project Overview

A custom website for Little Miss Violet Cafe, a real cafe located in Point Cook, Melbourne. The project aims to create an elegant online presence that reflects the cafe's brand and helps attract customers.

## Files

- `antigravity-modified.html` - Modified version with antigravity styling effects
- `little-miss-violet1.html` - Original version 1 of the website

## Features

- Elegant design with custom color scheme
  - Cream, parchment, blush, violet, and sage color palette
- Responsive layout for mobile and desktop
- Menu sections and cafe information
- Professional typography using Cormorant Garamond and DM Sans fonts
- Modern CSS with custom properties for theming

## Design Elements

- **Color Scheme**: Soft, elegant palette with violet as primary brand color
- **Typography**: Mix of serif (Cormorant Garamond) and sans-serif (DM Sans) fonts
- **Layout**: Responsive design optimized for various screen sizes
- **Styling**: Custom CSS with modern design principles

## Technology Stack

- HTML5
- Custom CSS with CSS variables
- Google Fonts API
- Responsive design principles

## Client Details

- **Business Type**: Cafe/Coffee Shop
- **Location**: Point Cook, Melbourne, Australia
- **Target Audience**: Local customers, cafe visitors
