# Cascade Memory System

A living knowledge base of everything learned about the user.

## Structure

- `preferences.md` - User preferences and workflow habits
- `projects.md` - Active and past projects with context
- `communication.md` - Communication style and patterns
- `technical.md` - Tech stack knowledge and preferences
- `facts.md` - Random facts, goals, interests

## Usage

Add new discoveries as they come up. Update existing entries when things change.
