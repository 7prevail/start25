# ClearSite Portfolio Project

**Status**: Active web design company showcase  
**Client**: ClearSite (internal company project)  
**Purpose**: Portfolio website showcasing web design services for local businesses  

## Project Overview

ClearSite is a web design company that creates websites for local business clients. This portfolio site demonstrates the company's capabilities and serves as a marketing tool to attract new customers.

## Files

- `clearsite-portfoliow favicon.html` - Main portfolio website with integrated favicon
- `clearsite-web-design-fixed.zip` - Fixed version backup
- `clearsite-web-design-safe.zip` - Safe version backup  
- `clearsite-web-design-updated.zip` - Updated version backup
- `d65fdb9a-62d0-43aa-98f8-46ee092185ea.png` - Portfolio screenshot or asset

## Features

- Modern responsive design using Tailwind CSS
- Custom gradient branding with blue to purple theme
- Professional typography using Clash Display and Satoshi fonts
- Integrated favicon with lightning bolt design
- Portfolio sections showcasing client work

## Technology Stack

- HTML5
- Tailwind CSS (via CDN)
- Custom CSS variables for theming
- Fontshare API for typography
- SVG favicon integration
