# start25

A minimal study companion built to help you organize subjects, track progress, and stay focused. No bloat, just what you need to get work done.

---

## Features

- **Subject Management** — Organize your study materials with tags and categories
- **Study Guides** — Generate personalized improvement plans with AI
- **Flashcards & Quiz** — Built-in review tools to test your knowledge
- **Progress Tracking** — XP, levels, and streaks to keep you motivated
- **Leaderboard** — Compete with other students (optional)
- **Dark Mode** — Easy on the eyes during late-night sessions
- **Accent Colors** — Personalize your theme

---

## Tech

- Vanilla HTML / CSS / JavaScript
- Firebase (leaderboard & username registry)
- Groq API (AI study guide generation)
- Google Sign-In
- Vercel hosting

---

## Local Development

Open `index.html` in any browser. No build step required.
